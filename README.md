# PacmanJV
A JAVA project using libgdx

## Project made during school
##### Objective
- Create a fully functional pacman

##### Commands
- Z : UP
- Q : LEFT
- S : DOWN
- D : RIGHT
- M : Mute sound
- ENTER : Start a new game ( sounds will superpose - v0.2.1 )

##### Score (default)
- Pellet -> 12pts
- Super Pellet -> 500pts
